# Release Checklist

1. Update `VERSION`.
2. Update `CHANGELOG.md`.
3. Update `RELEASE_NOTES.md`.
4. Run tests.
5. Build app locally.
6. Test installer.
7. Test portable ZIP.
8. Test update checker.
9. Test update install from previous version.
10. Confirm user data is preserved.
11. Confirm passwords are masked.
12. Push tag `vX.Y.Z`.
13. Confirm GitHub Actions release passed.
14. Download release files from GitHub.
15. Install on a clean Windows machine.
16. Verify app starts.
17. Verify ARK ASA profile loads.
18. Verify Palworld profile loads.
19. Publish release announcement.
