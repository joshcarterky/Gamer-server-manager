# Documentation

This folder contains user and developer documentation for Nexus Server Manager.

## Documents

- `INSTALL.md` - installation and startup instructions
- `CONFIGURATION.md` - runtime data, profiles, logs, backups, and server paths
- `release-preparation.md` - release process notes

Contributor-facing documents live at the repository root:

- `CONTRIBUTING.md`
- `CHANGELOG.md`
- `RELEASE_CHECKLIST.md`
